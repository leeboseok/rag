1.1 텍스트 임베딩
    # 인공지는 모델은 수많은 숫자로 이루어진 벡터를 사용하여 데이터를 처리하고 학습한다.
    # 텍스트를 수치화된 벡터로 변환하여 이를 분석(임베딩)
    # 워드 임베딩: 단어를 벡터로 변환
    # 문장 임베딩, 문서 임베딩
    # [1, 0.3, 0.5]와 같은 실수 벡터로 인코딩되어 이렇게 변환된 벡터를 통해 텍스트의 의미를 분석, 활용
    # 장점:
    # 1. 데이터를 수치화하여 처리할 수 있다.
    # 2. 벡터간 유사도 계산을 통해 텍스트의 문맥을 파악할 수 있음
1.2 코사인 유사도
    # 두 벡터 간의 각도를 통해 유사성을 측정하는 방법
    # 벡터의 크기가 아닌 방향성에 초점, -1에서 1 사이의 값을 가짐
    # 1: 두 벡터가 동일한 방향을 향함
    # 0: 두 벡터가 서로 직각을 이룸
    # -1: 두 벡터가 정반대 방향을 향함
    # 두 벡터의 방향성을 기반으로 유사성 평가
    # 데이터 크기와 무관하게 텍스트나 문서의 의미적 유사성 측정
    # 텍스트 분석, 문서 비교, 추천 시스템 등 활용
    # 실습1
