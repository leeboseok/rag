2. 문서 로더
    # 랭체인의 문서 로더는 다양한 데이터 소스와 형식을 랭체인 프레임워크에서 사용할 수 있도록
    # 표준화된 문서 형식으로 변환하는 도구.
    # 모든 로더는 데이터를 content와 metadata를 포함하는 표준 문서 객체로 변환하여
    # 랭체인의 다른 컴포넌트와 쉽게 통합할 수 있습니다.
    # [주요 문서 로더]
    # WebBaseLoader: 웹 페이지의 내용을 로드하는 기능 제공
    # PyPDFLoader: 파일을 로드하고 필요에 따라 페이지 단위로 분할하는 기능 제공
    # CSVLoader: CSV파일을 로드하는 기능 제공
    # DirectoryLoader: 지정된 디렉터리 내의 모든 파일을 로드하는 기능 제공
    # UnstructuredLoader: 다양한 형식의 파일을 로드하는 기능 제공
2.1 웹 페이지 로더
    # 랭체인은 웹 페이지의 HTML 콘텐츠를 로드하고 파싱하여 문서 형식으로 변환하는 다양한 도구를 제공
    # WebBaseLoader
    # 단일 또는 여러 웹 페이지에서 텍스트 로드 가능
    # BeautifulSoup활용 HTML을 세부적으로 파싱할 수 있음
    # 비동기 로딩과 지연 로딩으로 성능을 높일 수 있음
    # 단일 웹페이지는 String 타입, 다중 웹 페이지는 리스트 형식 전달 처리할 수 있음
    # 데이터 로드 결과 리스트로 변환, 각 웹 페이지의 메타데이터(출처, 제목, 언어 등)와 내용 확인할 수 있음
2.2 PDF 로더
    # PyPDFLoader
    # PDF 문서를 페이지 단위로 로드하는 데 자주 사용하는 도구
    # 페이지를 청크 단위로 나누어 로드
    # 하지만, 텍스트가 없이 이미지만 있거나 빈페이지일 경우 제외될 수 있음
    # 간단, 사용 쉬움, 복잡한 레이아웃의 PDF에서는 정확도 떨어질 수 있음
    # PyMuPDFLoader: PyPDFLoader보다 빠르고 정확한 텍스트 추출이 가능, 이미지나 표, 레이아웃 정보 등 추출 가능
    # 빠른 속도, 다양한 메타데이터 추출 기능 제공
    # 다수, 대용량 PDF 처리에 효과적
    # PDFPlumberLoader: 표와 같은 구조화된 데이터 추출에 특화
    # 다양하고 상세한 메타데이터 제공
    # PDF 문서의 구조와 내용을 깊이 이해하는데 유리
    # 상세한 분석을 수행하기 때문에 PyMuPDFLoader에 비해 처리 속도가 다소 느림. 
2.3 CSV 로더
    # CSV: 데이터를 행과 열고 구성하여 저장하는 텍스트 파일 형식
    # 다양한 데이터 분석과 처리 작업에 사용
    # CSVLoader
    # CSV파일을 행 단위로 로드하여 각 행을 독립적인 문서로 변환
    # UnstructuredCSVLoader
    # CSV파일을 유연하게 처리할 수 있는 도구
